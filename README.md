## Instructions

1. Enter the repo directory.

```
cd nft_ranking/
```

2. Set up and activate virtual environment.

```
[[ -d env ]] || python3 -m venv env
source env/bin/activate
```

3. Install packages.

```
python3 -m pip install -r requirements.txt
```

4. Run the script from the virtual environment.

```
python3 -m runner
```
